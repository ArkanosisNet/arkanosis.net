Sélection d'extensions pour Mozilla Firefox 2.0.0.10 (Microsoft Windows XP SP2)

Dernière mise à jour le 27 Novembre 2007

Introduction

Cette page regroupe les extensions pour Mozilla Firefox que j'utilise.

Elles sont regroupées en cinq catégories :

Extensions indispensables
Je ne peux plus concevoir d'utiliser Mozilla Firefox sans ces extensions ; ou c'est très difficile.
Extensions presque indispensables
Je pourrais me passer de ses extensions, mais je préfère vraiment les avoir.
Extensions gadget
Ces extensions n'ont pas de réel intérêt. Je les garde surtout pour le fun... ou la frime.
Extensions pour développeurs
Ces extensions sont indispensables pour les développeurs web. Pour les autres, elles sont inutiles.
Extensions pour extensions
Quand on commence à avoir beaucoup d'extensions, il devient difficile de s'y retrouver. Ces extensions permettent de mieux gérer les extensions de Mozilla Firefox.
